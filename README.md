# 🚦 Movilidad urbana y productividad económica

Este proyecto analiza la relación entre la **movilidad urbana** (congestión vehicular, tiempos de viaje) y la **productividad económica** (PIB per cápita). El objetivo es identificar patrones y posibles ciudades prioritarias para inversión en infraestructura de transporte.

---

## 🧾 Resumen ejecutivo

**Contexto & objetivo:**  
El análisis busca responder: ¿qué relación existe entre la congestión vehicular y los indicadores de productividad económica? Se utilizan variables de tráfico urbano y PIB per cápita para evaluar correlaciones y tendencias relevantes para la toma de decisiones.

**Metodología (alto nivel):**  
- Limpieza y estandarización de datos.  
- Agregación por ciudad–año.  
- Unión INNER entre tráfico y economía.  
- Validaciones visuales: distribuciones, outliers y tendencias generales.  

**Hallazgos iniciales:**  
- Se observan patrones donde altos niveles de congestión se asocian con menor productividad.  
- Bogotá y Lima muestran correlaciones significativas que sugieren impacto económico negativo.  
- Buenos Aires presenta anomalías que requieren análisis adicional.  

**Recomendaciones:**
